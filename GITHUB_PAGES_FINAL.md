## 💡 Solução de Problemas

**Página em branco?**
- Aguarde 2-3 minutos
- Hard refresh: Ctrl+Shift+R no navegador
- Verifique se `/docs` folder está selecionado nas Settings

**Apenas README aparece?**
- Confirme que selecionou `/docs` (não `/root`)
- Verifique se `index.html` está em `/docs/`

**Arquivos não carregam?**
- Confirme que `.nojekyll` existe em `/docs/`
- Verifique se os caminhos relativos estão corretos
